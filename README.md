# Proyecto1 Compiladores I

_Práctica 1, donde se desarrolla una aplicacion con la estructura cliente servidor._

_La primera aplicación es una aplicacion hecha con Kotlin que sirve como cliente, la cual se conecta con la ayuda de sockets a una aplicacion hecha con Java que sirve como servidor._

_El cliente de Kotlin sirve para recibir la instruccion de Mapa y Movimiento._

_El servidor Java recibe los mapas en formato JSON para analizarlos con la ayuda de JFlex y Cup para posteriormente enviar una respuesta en formato XML al cliente._

_Se aplicaron conocimientos como, expresiones regulares, gramaticas libres de contexto, sin recursividad por la izquierda, precedencia, etc. _

_Hecho con Java, Kotlin, JFlex y Cup._
